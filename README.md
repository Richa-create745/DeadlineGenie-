
# 🚀 DeadlineGenie - AI Productivity Companion

## 📌 Overview

DeadlineGenie is a smart productivity companion that helps users organize daily tasks, track goals, manage habits, and improve productivity through intelligent task recommendations.

The application allows users to prioritize tasks, monitor progress, and receive smart suggestions based on task priority and deadlines.

---

## ✨ Features

### 📋 Task Management

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Store tasks using Local Storage
* Deadline management
* Priority management (High, Medium, Low)

### 🔍 Search & Filter

* Search tasks instantly
* Filter tasks by:

  * All Tasks
  * High Priority
  * Medium Priority
  * Low Priority
  * Completed Tasks

### 📂 Categories

Organize tasks into categories:

* 📚 Study
* 💼 Work
* 🏠 Personal
* 🏃 Health

### 📊 Dashboard

* Total Tasks
* High Priority Tasks
* Total Goals
* Total Habits
* Dynamic Progress Bar

### 🎯 Goals

* Add goals
* Delete goals
* Local storage support

### 🔥 Habits

* Add daily habits
* Delete habits
* Local storage support

### 💡 Smart Productivity Recommendations

The recommendation engine:

* Sorts tasks by priority
* Detects upcoming deadlines
* Warns about overdue tasks
* Ignores completed tasks
* Displays a congratulatory message when all tasks are completed
